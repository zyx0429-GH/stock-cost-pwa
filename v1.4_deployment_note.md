# v1.4 全內聯版部署 - 2026-06-25

## 問題根源
v1.3.3 手機版卡「正在抓取資料」的連環 bug：
1. **`/app.js` 和 `/style.css` 被 SPA catch-all 攔截成 HTML**，JavaScript 根本沒載入
2. **Service Worker 快取了 `'/'` 頁面**，Render 冷啟動時還原舊版 HTML
3. **CSS `.loading { display: none !important; }` 永遠蓋過 `.loading.active`**，載入動畫永遠看不到

## v1.4 解決方案
- **完全內聯**：CSS 和 JS 全部寫在 `index.html` 內，零外部靜態資源依賴（除了 CDN chart.js）
- **移除 Service Worker 註冊**：不再掛 `/sw.js`，從根本上杜絕快取問題
- **no-cache middleware**：所有非 API 路徑強制 Cache-Control: no-cache, no-store, must-revalidate
- **45 秒 AbortController 超時**：不會無限期卡住
- **冷啟動進度提示**：5 秒後顯示「伺服器冷啟動中」，15 秒後顯示「仍在計算中」
